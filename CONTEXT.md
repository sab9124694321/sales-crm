# SZB CRM — Контекст проекта

## О проекте
CRM-система для отдела продаж эквайринга Сбербанка.
Управление продажами и сопровождение эквайринга.
Стек: PHP + SQLite + GigaChat AI (Сбер)
Хостинг: Docker на VPS (5.129.248.239)

## Команда
- Работаем отделами продаж эквайринга Сбербанка
- "Охотники" — посторонние люди (внешние агенты), приводящие лиды
- Мы — внутренние менеджеры

## Ограничения безопасности (критично!)
| Ограничение | Почему | Как обходим |
|-------------|--------|-------------|
| Нельзя хранить данные клиентов на внешних серверах | Запрет безопасности Сбера | Используем почту, планируем интеграции |
| Нет доступа к микрофону с рабочего ноутбука | Политика безопасности | Анализ голоса — только на Android |
| Не работает кнопка отправки почты на корпоративных iPad | Техническое ограничение | Ищем обходные пути |
| Telegram заблокирован в России | Регуляторное ограничение | Планировали автоотчёты в Макс и на почту |

## Технологии
| Компонент | Выбор | Почему |
|-----------|-------|--------|
| AI | GigaChat (Сбер) | Бесплатные токены + российское происхождение |
| База данных | SQLite | Простота, не требует отдельного сервера |
| Хостинг | Docker на VPS | Изоляция, контроль |

## Роли пользователей
| Роль | Кто это |
|------|---------|
| manager | Менеджер по продажам |
| head | Начальник малого бизнеса |
| territory_head | Начальник управления безналичных решений |
| terman | Терминальный менеджер |
| admin | Администратор |
| mmb_manager / mmb_tp_head | ММБ (малый и микро бизнес) |
| ubr_middle | Менеджер УБР с частично уменьшенным планом (помогает ММБ) |

## Термины
- УБР — управление безналичных решений (основные пользователи)
- ММБ — подразделение малого и микро бизнеса
- ТЭ — торговый эквайринг
- ТСТ — торговая точка
- Ритм — платформа для задач (бывш. Тортуга)
- Макс — российский национальный мессенджер (интеграция работает, тестовое сообщение ушло)
- Охотники — внешние агенты, приводящие лиды
